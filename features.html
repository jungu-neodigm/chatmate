Product Features

AI 챗봇 운영에 필요한 기능을
하나의 플랫폼에서 제공합니다.

지식 데이터 연결부터 답변지침, 추천질문, 질문 리스트, 사용량 관리, 개인정보 보호까지
AI 챗봇을 실제 서비스로 운영하기 위한 기능을 제공합니다.

01 · 지식 데이터 관리

웹사이트, 문서, FAQ, 상품 정보를 연결합니다.

웹사이트, 사이트맵, FAQ, 문서, 상품 정보, 매뉴얼 등 챗봇이 답변할 지식 데이터를 연결하고 관리합니다. 초기에는 URL, 사이트맵, 직접 Q&A 등록으로 빠르게 시작하고, 이후 문서 업로드와 외부 시스템 연동으로 확장할 수 있습니다.

  • ✓사이트맵 URL 등록
  • ✓직접 Q&A 등록
  • ✓URL별 수집 상태 관리
  • ✓사용/사용중지 설정
  • ✓우선순위 설정
  • ✓실패 사유 표시 및 재처리

데이터 처리 상태

등록됨대기
수집중 → 처리중진행
완료✓
실패 → 재처리중확인 필요

기본 답변 정책

  • ✓내부 지식 데이터 우선 검색
  • ✓고객사/프로젝트별 데이터 분리
  • ✓출처 기반 답변
  • ✓근거 부족 시 안전한 fallback
  • ✓답변 범위 제한 & RAG 실패 기록

02 · RAG 기반 답변

근거 없이는 답변하지 않습니다.

AI가 임의로 답변하지 않도록 고객사가 등록한 지식 데이터를 먼저 검색하고, 검색된 근거를 기반으로 답변을 생성합니다. 근거가 부족한 경우에는 무리하게 답변하지 않고 재질문, 추천질문, 상담 연결 등 안전한 fallback을 제공합니다.

03 · 추천질문 관리

첫 화면부터 자연스러운 대화 시작.

첫 화면이나 답변 실패 상황에서 사용자가 쉽게 질문을 시작할 수 있도록 추천질문을 제공합니다. 운영자는 프로젝트별로 추천질문을 등록하고 관리할 수 있습니다.

  • ✓첫 화면 추천질문
  • ✓RAG 실패 시 추천질문
  • ✓추천질문 클릭 시 즉시 질문 실행
  • ✓질문 리스트 기반 추천질문 등록

04 · 답변지침 관리

브랜드 톤과 응대 기준을 프로젝트별로 설정합니다.

챗봇이 어떤 말투로, 어떤 범위까지, 어떤 기준으로 답변해야 하는지 설정합니다. 기업의 브랜드 톤, 공공기관의 공식 톤, 병원의 신중한 안내 톤 등 업종별 응대 기준을 적용할 수 있습니다.

업종톤
이커머스친절하고 구매 결정을 돕는 톤
SaaS명확하고 전문적인 설명 톤
병원신중하고 안내 중심의 톤
교육쉬운 설명과 단계별 안내 톤
금융보수적이고 규정 준수 중심 톤
공공기관공식적이고 간결한 존댓말 톤
내부 업무빠르고 실무적인 안내 톤

05 · 질문 리스트 및 대화 로그

고객이 실제로 무엇을 묻는지 확인하세요.

답변 실패나 부정 평가가 발생한 질문을 개선할 수 있습니다. 질문 리스트는 AI 챗봇 운영의 핵심 개선 도구입니다.

질문 상태

정상답변 / RAG실패 / 부정평가 / 개인정보차단

처리 상태

미처리 / 검토중 / 답변등록완료

필터

키워드 / 기간 / 카테고리

기록

세션 ID, 질문 횟수, 마지막 발생일

06 · 상담/예약/구매 CTA 연결

답변에서 끝나지 않고, 다음 행동으로 연결합니다.

정보 안내에서 비교, 추천, 상담 연결, 예약, 신청까지 고객의 다음 행동을 자연스럽게 연결합니다. 관심도가 높은 방문자는 데모 예약, 상담 신청, 구매 링크로 이어집니다.

07-10 · 운영 관리 기능

운영에 필요한 나머지 기능도 빠짐없이.

사용량 관리

프로젝트별 토큰 사용량과 비용 한도를 관리하고, 한도 초과 시 AI 생성을 제한합니다.

권한 관리

운영사/고객사 관리자, 운영 담당자 권한을 분리하고 접근 범위를 제한합니다.
